# 个人前端项目
个人练习...，类似论坛类网站...

## 前端：
- vue

## 后端：
- node.js
- express
- mysql

## 运行：
前后端分别安装依赖包：
```bash
pnpm i
```
启动前端服务：
```bash
pnpm dev
```
启动后端服务：
```bash
node server.js
```
